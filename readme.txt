A simple job board on your WordPress website where admin can post job requirements. Also, applicants can apply to a job in an easy and elegant way.

== Description ==
admin can add jobs with category, skills, occupation, timing of the job etc. and applicant can apply for a job in a simple form.



Features:

1. Add job
2. Job with category, skills, occupation, timing etc
3. Apply form in job detail
4. E-mail notification
5. Manage job application in admin side




== Installation ==
1. Upload "job-board-vanila" to the "/wp-content/plugins/" directory.
2. Activate the plugin through the "Plugins" menu in WordPress.
